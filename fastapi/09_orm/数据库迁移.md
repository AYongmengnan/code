### 安装aerich
```shell
pip intsall aerich
```
## 使用
### 1.初始化配置，只需使用一次
```shell
aerich init -t settings.TORTOISE_ORM  # TORTOISE_ORM配置位置
```
![!\[alt text\](image.png)](img/1.png)
### 2.初始化数据库，一般情况下只用一次
```shell
aerich init-db
```
![alt text](<img/2.png>)
### 3.更新模型并进行迁移
修改model类，重新生成迁移文件
```shell
aerich migrate --[name] # 标记修改操作,aerich migrate --name add field addr
```
![alt text](img/3.png)
#### a.升级
```shell
aerich upgrade
```
![alt text](img/4.png)
#### b.降级
```shell
aerich downgrade
```
