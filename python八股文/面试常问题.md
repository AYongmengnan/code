### Lambda表达式
``` txt
Lambda表达式是一种匿名函数,可以接受任意多个参数（包括可选参数）并且返回单个表达式值得函数。
```
```python
add = lambda x, y: x + y
print(add(3, 5))  # 输出：8
```
